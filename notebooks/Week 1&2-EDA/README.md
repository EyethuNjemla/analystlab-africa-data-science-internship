# Week 1–2: Data Cleaning & Exploratory Data Analysis

## Objective

The objective of this project is to prepare two real-world datasets for machine learning by performing data cleaning and exploratory data analysis (EDA).

## Datasets

- Titanic Dataset
- House Prices Dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks

- Load datasets
- Inspect data structure
- Handle missing values
- Remove duplicates
- Explore feature distributions
- Analyze relationships between variables
- Generate visualizations
- Summarize insights

# Key Findings

## Titanic Dataset

- Female passengers had a higher survival rate than male passengers.
- Passenger class influenced survival outcomes.
- Higher ticket fares were associated with increased survival.
- Most passengers were between 20 and 40 years old.

## Housing Dataset

- Larger houses generally had higher selling prices.
- Houses with more bedrooms tended to cost more.
- Property area showed a strong positive relationship with price.
- Some high-priced properties may be considered outliers.
